# ToDo App

## 1. Link and Preview

Project link is available at [ToDo App](https://aligoknl.github.io/ToDo-App/client/index.html)

![Preview of ToDo App!](/client/public/images/todo.GIF)

## 2. Description

The ToDo App is a full stack web application that helps you to remind your tasks. All data are stored in the database. You can add, update, delete and search a task.

## 5. Must Haves

-   Data should come from a database.
-   A user should be able to search a task name and see related results.
-   A user should be able to add a task and the task must be added above the list.
-   A user should be able to update a task.
-   A user should be able to delete a task.

## 6. Nice-to Have

-   A user can refresh the page.
